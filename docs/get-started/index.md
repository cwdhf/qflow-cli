# Get started with Qflow CLI

Welcome to Qflow CLI! This guide will help you install, configure, and start
using the Qflow CLI to enhance your workflow right from your terminal.

## Quickstart: Install, authenticate, configure, and use Qflow CLI

Qflow CLI brings the power of advanced language models directly to your command
line interface. As an AI-powered assistant, Qflow CLI can help you with a
variety of tasks, from understanding and generating code to reviewing and
editing documents.

## Install

The standard method to install and run Qflow CLI uses `npm`:

```bash
npm install -g @google/gemini-cli
```

Once Qflow CLI is installed, run Qflow CLI from your command line:

```bash
gemini
```

For more installation options, see [Qflow CLI Installation](./installation.md).

## Authenticate

To begin using Qflow CLI, you must authenticate with a Google service. In most
cases, you can log in with your existing Google account:

1. Run Qflow CLI after installation:

   ```bash
   gemini
   ```

2. When asked "How would you like to authenticate for this project?" select **1.
   Login with Google**.

3. Select your Google account.

4. Click on **Sign in**.

Certain account types may require you to configure a Google Cloud project. For
more information, including other authentication methods, see
[Qflow CLI Authentication Setup](./authentication.md).

## Configure

Qflow CLI offers several ways to configure its behavior, including environment
variables, command-line arguments, and settings files.

To explore your configuration options, see
[Qflow CLI Configuration](./configuration.md).

## Use

Once installed and authenticated, you can start using Qflow CLI by issuing
commands and prompts in your terminal. Ask it to generate code, explain files,
and more.

To explore the power of Qflow CLI, see [Qflow CLI examples](./examples.md).
